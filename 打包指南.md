# Lumina Library 打包指南

## 一、安装前置工具

### 1. Node.js（已装则跳过）
下载安装：https://nodejs.org （选 LTS 版本）

### 2. Rust 编译器
```powershell
# PowerShell 中执行，一路回车选默认
Invoke-WebRequest -Uri https://win.rustup.rs/ -OutFile rustup-init.exe
.\rustup-init.exe
```
装完后**重启终端**。

### 3. WebView2（Win11 自带，Win10 需安装）
https://developer.microsoft.com/en-us/microsoft-edge/webview2/

---

## 二、打包 Windows 桌面版

```powershell
cd D:\proj_ebook

# 安装 Tauri CLI
npm install

# 生成图标（先打开 generate-icons.html 下载图片到 src-tauri/icons/）
# 或者用 Tauri 官方图标工具：
# npm install -g @tauri-apps/cli
# npx @tauri-apps/cli icon src-tauri/icons/icon.png

# 开发模式运行（测试用）
npm run dev

# 正式打包（首次编译需下载依赖，约5-10分钟）
npm run build
```

打包产物在：
```
src-tauri/target/release/bundle/
├── msi/          ← Windows 安装包（.msi）
└── nsis/         ← Windows 安装包（.exe，更小）
```

## 三、打包 Android 版

### 1. 安装 Android Studio
https://developer.android.com/studio

安装时勾选：
- Android SDK
- Android SDK Platform
- Android Virtual Device

### 2. 配置环境变量
```
ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk
JAVA_HOME = Android Studio 内置 JDK 路径
```
把以下路径加入 PATH：
```
%ANDROID_HOME%\platform-tools
%ANDROID_HOME%\emulator
%ANDROID_HOME%\tools
%ANDROID_HOME%\tools\bin
%ANDROID_HOME%\cmdline-tools\latest\bin
```

### 3. 安装 Android SDK 组件
```powershell
sdkmanager "platforms;android-34"
sdkmanager "build-tools;34.0.0"
sdkmanager "ndk;27.0.12077973"
```

### 4. 初始化并打包
```powershell
cd D:\proj_ebook

# 初始化 Android 项目（首次）
npx tauri android init

# 打包 APK
npm run build:android
```

产物在：
```
src-tauri/gen/android/app/build/outputs/apk/universal/release/
```

## 四、更换应用图标

1. 准备一张 1024x1024 的 PNG 图标
2. 执行：
```powershell
npx @tauri-apps/cli icon src-tauri/icons/icon.png
```
这会自动生成所有平台需要的图标尺寸。

## 常见问题

**Q: Rust 下载很慢？**
A: 设置国内镜像，在环境变量中添加：
```
RUSTUP_DIST_SERVER=https://mirrors.tuna.tsinghua.edu.cn/rustup
RUSTUP_UPDATE_ROOT=https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup
```

**Q: Android NDK 下载失败？**
A: 在 Android Studio → Settings → SDK Manager → SDK Tools 中勾选 NDK 安装。

**Q: 打包后数据存在哪？**
A: 各平台的 localForage/IndexedDB 数据位置：
- Windows: `%APPDATA%/com.lumina.library/`
- Android: App 内部存储
