# yread Lib

一个基于 Tauri v2 的本地 PDF 阅读器，支持电子书与论文的管理、阅读、笔记批注。

## 功能特性

### 书架管理（电子书）
- **批量导入 PDF** — 支持多选导入，自动去重（书名 + 文件大小）
- **智能标签分类** — 根据文件名自动归类（技术、文学、历史、科学等），支持自定义标签
- **艺术封面生成** — 为无封面的书籍自动生成渐变封面
- **书名字号可调** — 鼠标悬停书本卡片，拖动滑块调整字号（12px ~ 48px）
- **书名可编辑** — 书籍详情页点击书名直接修改
- **一键去重** — 快速清除同名重复书籍

### 论文管理
- **独立页面** — 论文与电子书分开管理，拥有独立的标签分类体系
- **阅读状态跟踪** — 四种状态：未读 / 在读 / 已读 / 重读，支持按状态筛选
- **论文元数据** — 自由编辑作者、期刊/会议、年份、DOI、URL 等信息
- **论文速览** — 自动提取 Abstract、Figures（图表说明）、Conclusion，快速判断是否值得精读
- **荧光笔高亮** — 自由手绘荧光标注，笔刷大小可调，支持撤销/清除，点击高亮可删除

### PDF 阅读器
- **三种阅读模式** — 单页、双页、滚动模式自由切换
- **缩放控制** — 支持页面缩放
- **进度记忆** — 自动记录每本书的阅读进度
- **键盘快捷键** — 翻页（PageUp/PageDown）、缩放（+/-）、关闭（Esc）

### 笔记功能
- **实时预览** — 左右分栏，左侧编辑 Markdown，右侧实时渲染预览（类似 Obsidian）
- **数学公式** — 支持 KaTeX 渲染，`$$...$$` 块级公式，`$...$` 行内公式
- **手写批注** — 基于 Signature Pad 的手写画布，支持鼠标和触屏
- **浮动面板** — 笔记以毛玻璃浮动面板显示在 PDF 上方，可拖动位置、调整大小，不遮挡阅读内容
- **自动保存** — 输入即保存，关闭重开后保留所有笔记

### 阅读统计
- 藏书总数、累计阅读时长、阅读次数
- 书籍与论文分开统计：各自独立的阅读时长、阅读次数、标签分布
- 按周柱状图展示阅读时长趋势（书籍/论文分别展示）
- 标签分布饼图（书籍/论文分别展示）
- 阅读时长排行榜（书籍/论文分别展示）

### 其他
- **暗色/亮色主题** — 一键切换
- **全局搜索** — 可拖动的毛玻璃搜索按钮，模糊匹配书名和论文信息
- **数据本地存储** — 基于 IndexedDB，无需联网

## 技术栈

- **框架**: Tauri v2 (Rust + WebView2)
- **前端**: 原生 HTML/CSS/JavaScript，无构建工具
- **PDF 渲染**: PDF.js
- **Markdown**: marked.js
- **数学公式**: KaTeX
- **手写**: Signature Pad
- **图表**: Chart.js
- **存储**: localForage (IndexedDB)

## 开发环境

### 前置要求
- Node.js (LTS)
- Rust 编译器 (`rustup`)
- WebView2 (Win11 自带)

### 开发运行
```bash
npm install
npm run dev
```

### 打包
```bash
# 构建前端资源
cp index.html style.css app.js dist/
cp -r lib dist/

# 打包为 Windows 安装包
npm run build
```

产物位于 `src-tauri/target/release/bundle/nsis/`。

```
│   ├── lucide.min.js       # 图标

│   ├── Cargo.toml
│   └── src/
├── release/            # 发布产物
├── dist/               # 前端构建资源
└── package.json
```

## 更新日志

### v1.0.2 (最新)
- **热力图日期修复**：修复统计日期因 `toISOString()` UTC 时区偏移导致的日期错误（UTC+8 下 16:00 后日期偏差一天）
- **主题色系统**：设置页新增深色/浅色主题色选择器，5 种颜色（靛蓝、紫色、青色、翠绿、玫红），全局生效于按钮、热力图、图表、底纹、书脊等
- **彭罗斯底纹动画**：底纹不再静止，CSS 动画实现缓慢旋转与缩放漂移（180s 周期），暗主题下对比度大幅提升
- **透明度范围扩展**：底纹透明度滑块上限从 20% 扩展至 50%
- **毛玻璃特效**：设置卡片与统计卡片使用 `backdrop-filter: blur()` 模糊彭罗斯底纹
- **封面生成器增强**：纽结类型从 3 种扩展至 8 种（新增五瓣结、Torus(3,4)、Lissajous 结、玫瑰线、Torus(4,5)）；种子加入时间戳，每次生成不同；底部绘制区域裁剪避免曲线与标题重叠
- **书名叠加层**：封面标题区域渐变遮罩增强，text-shadow 双层描边，任何背景上均可清晰阅读
- **设置页修复**：「设置」标题不再被导航栏遮挡
- **PDF 加载修复**：禁用 Worker 线程（Tauri 自定义协议不支持 Worker），改用主线程解析，解决 PDF 加载失败问题
- **悬浮按钮组**：原搜索按钮改为 2×2 四宫格圆角按钮（搜索、计算器、两个待定），中心四角星装饰，整体可拖动，高透毛玻璃风格，随主题色变化
- **内置计算器**：支持简易模式（加减乘除、小数点、百分号、开方）与科学模式（sin/cos/tan/asin/acos/atan、log/ln、√、x^y、e^x、n!、|x|、1/x、π、e），模式一键切换

### v1.0.1
