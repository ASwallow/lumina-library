# yread Lib

一个基于 Tauri v2 的本地 PDF 阅读器，支持电子书与论文的管理、阅读、笔记批注。

## 功能特性

### 书架管理（电子书）
- **批量导入 PDF** — 支持多选导入，自动去重（书名 + 文件大小）
- **智能标签分类** — 根据文件名自动归类（技术、文学、历史、科学等），支持自定义标签
- **艺术封面生成** — 为无封面的书籍自动生成渐变封面
- **书名字号可调** — 鼠标悬停书本卡片，拖动滑块调整字号（12px ~ 48px）
- **书名可编辑** — 书籍详情页点击书名直接修改
- **一键去重** — 快速清除同名重复书籍

### 论文管理
- **独立页面** — 论文与电子书分开管理，拥有独立的标签分类体系
- **阅读状态跟踪** — 四种状态：未读 / 在读 / 已读 / 重读，支持按状态筛选
- **论文元数据** — 自由编辑作者、期刊/会议、年份、DOI、URL 等信息
- **论文速览** — 自动提取 Abstract、Figures（图表说明）、Conclusion，快速判断是否值得精读
- **荧光笔高亮** — 自由手绘荧光标注，笔刷大小可调，支持撤销/清除，点击高亮可删除

### PDF 阅读器
- **三种阅读模式** — 单页、双页、滚动模式自由切换
- **缩放控制** — 支持页面缩放
- **进度记忆** — 自动记录每本书的阅读进度
- **键盘快捷键** — 翻页（PageUp/PageDown）、缩放（+/-）、关闭（Esc）

### 笔记功能
- **实时预览** — 左右分栏，左侧编辑 Markdown，右侧实时渲染预览（类似 Obsidian）
- **数学公式** — 支持 KaTeX 渲染，`$$...$$` 块级公式，`$...$` 行内公式
- **手写批注** — 基于 Signature Pad 的手写画布，支持鼠标和触屏
- **浮动面板** — 笔记以毛玻璃浮动面板显示在 PDF 上方，可拖动位置、调整大小，不遮挡阅读内容
- **自动保存** — 输入即保存，关闭重开后保留所有笔记

### 阅读统计
- 藏书总数、累计阅读时长、阅读次数
- 书籍与论文分开统计：各自独立的阅读时长、阅读次数、标签分布
- 按周柱状图展示阅读时长趋势（书籍/论文分别展示）
- 标签分布饼图（书籍/论文分别展示）
- 阅读时长排行榜（书籍/论文分别展示）

### 其他
- **暗色/亮色主题** — 一键切换
- **全局搜索** — 可拖动的毛玻璃搜索按钮，模糊匹配书名和论文信息
- **数据本地存储** — 基于 IndexedDB，无需联网

## 技术栈

- **框架**: Tauri v2 (Rust + WebView2)
- **前端**: 原生 HTML/CSS/JavaScript，无构建工具
- **PDF 渲染**: PDF.js
- **Markdown**: marked.js
- **数学公式**: KaTeX
- **手写**: Signature Pad
- **图表**: Chart.js
- **存储**: localForage (IndexedDB)

## 开发环境

### 前置要求
- Node.js (LTS)
- Rust 编译器 (`rustup`)
- WebView2 (Win11 自带)

### 开发运行
```bash
npm install
npm run dev
```

### 打包
```bash
# 构建前端资源
cp index.html style.css app.js dist/
cp -r lib dist/

# 打包为 Windows 安装包
npm run build
```

产物位于 `src-tauri/target/release/bundle/nsis/`。

│   ├── Cargo.toml

├── release/            # 发布产物

- **笔记封面颜色**：笔记编辑器新增颜色选择器，7 种主题色可选，卡片顶部显示颜色条
- **笔记卡片液态玻璃**：笔记卡片升级为液态玻璃效果（blur(20px) + saturate(1.4)），亮主题下彭罗斯底纹对比度提升至 18%，增强毛玻璃透出感

### v1.0.5
- **笔记关系图谱分析**：统计页新增知识图谱分析卡片（毛玻璃风格），自动检测连通分量（独立知识网），展示节点数、边数、平均度（每节点平均连边数）、独立环数（圈秩）、最短环长度等指标，支持按子图筛选查看

### v1.0.4
- **独立笔记Tab**：笔记作为单独 Tab（与书架/论文/统计/设置并列），支持独立创建、编辑、删除笔记，Markdown + 手写
- **差分阅读模式**：PDF 与 PDF 差分阅读时不允许打开笔记，PDF 与笔记差分阅读时也不允许打开笔记，两部分有独立的位置和缩放比例控制，笔记可编辑
- **笔记双链与关系图**：笔记间支持 `[[笔记名]]` 双链语法，生成 Obsidian 风格的力导向图动画，图节点大小、颜色、斥力水平可在设置中调整，边颜色和粗细可配置
- **LaTeX 公式自动补全**：Tab 键触发/确认，方向键导航，支持希腊字母、运算符、关系符、箭头、函数、符号、标注、括号、矩阵等 80+ 条目，KaTeX 渲染公式预览
- **琼斯多项式封面生成器**：以书名+时间戳为种子生成伪琼斯多项式参数，三种类型：纽结（10 种变体）、拓扑（环面/克莱因瓶/莫比乌斯/球极投影/Boy 曲面）、几何（分形树/阿基米德螺旋/对称镶嵌/谢尔宾斯基/李萨如）
- **设置页液态玻璃特效**：彭罗斯底纹可透过半透明毛玻璃卡片显示，backdrop-filter 模糊+饱和度增强
- **计算器溢出修复**：科学模式切换时自动重新定位不超出视口，calc-panel z-index 提升至 200
- **阅读器 FAB 按钮组**：阅读时可呼出四宫格悬浮按钮（搜索自动禁用），z-index 180 高于阅读器 150
